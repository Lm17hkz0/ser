Bem-vindo ao Seu Projeto de Alma Gêmea!

Esta é uma aplicação web interativa construída com Next.js e React.

Para visualizar a aplicação completa com todas as funcionalidades (quiz, chat, imagens e áudio), você precisa iniciá-la como um projeto Next.js.

Como iniciar o projeto:

1. Abra o terminal ou prompt de comando na pasta raiz do projeto.

2. Instale as dependências (se ainda não o fez):
npm install

3. Inicie o servidor de desenvolvimento:
npm run dev

4. Após iniciar, a aplicação estará disponível em http://localhost:3000 (ou outra porta indicada no terminal).

Ir para a Aplicação (se já estiver rodando)

Este arquivo `index.html` é uma página estática. A aplicação principal é dinâmica e requer o Next.js para ser executada.
